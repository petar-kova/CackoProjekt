# Akril Art Shop (.NET MVC + Web API)

Profesionalna web trgovina za prodaju akrilnih boja i slikarskog pribora, izrađena u .NET tehnologijama.

## Implementirano
- ASP.NET Core MVC aplikacija (frontend + backend)
- Login i registracija putem ASP.NET Core Identity
- Početna stranica s akcijama i najprodavanijim proizvodima
- Stranica svih proizvoda sa sortiranjem i filtriranjem po vrsti
- Web API endpoint: `/api/products`
- SQLite baza sa seed podacima

## Pokretanje
```bash
dotnet restore
dotnet run
```
